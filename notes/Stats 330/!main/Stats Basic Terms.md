### example 
- $\Omega$ = {1,2,3,4,5}
- A = {1,2,3}
- B = {2,3,4}
- C = {4,5}
	- $\overline{A}$ = {4,5}
	- A $\cup$ B = {1,2,3,4}
	- A $\cap$ B = {2,3}
	- A $\cap$ C = $\emptyset$
	- A and B are not disjoint
	- A and C are disjoint
	- A, B, and C are not pairwise disjoint
